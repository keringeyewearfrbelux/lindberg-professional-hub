LINDBERG Professional Hub — GitHub Pages package

Purpose:
A desktop-first and mobile-responsive GitHub Pages landing page for a professional email signature.
No ambassador/contact section is included.

Files to upload at the root of a NEW GitHub repository:
- index.html
- assets/
- README.txt

IMPORTANT VIDEO STEP:
Add your 1080p video file into the assets folder and name it exactly:
Lindberg Video.mp4

Current page behavior:
- Full-screen video hero with centered text, inspired by the LINDBERG website campaign hero.
- If the video is missing or loading slowly, the page displays assets/lindberg-video-poster.jpg.
- Three transparent buttons:
  1. Train yourself with Kering Eyelearn
  2. Access to your B2B platform
  3. Communicate with our assets

After upload to GitHub Pages:
1. Create a new public repository, e.g. lindberg-professional-hub.
2. Upload index.html, assets/, README.txt.
3. Activate GitHub Pages from Settings > Pages > Deploy from branch > main > /root.
4. Test on desktop and iPhone before adding to Outlook signature.
